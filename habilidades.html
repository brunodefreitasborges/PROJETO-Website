HTML5 e CSS3


Iniciei este site como meu primeiro passo em webdesign. Ele conta com meta tags de conteúdo para facilitar o trabalho dos indexadores do Google. O foco do design é ser de fácil leitura e nenhuma poluição visual. Também me preocupei com a velocidade de acesso, manipulando imagens para serem do menor tamanho possíveis.


Meu software de escolha para este tópico é o Visual Studio Code.
